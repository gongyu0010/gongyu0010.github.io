机器学习基础

机器学习

  • 机器学习是一门多领域交叉学科，涉及概率论、统计学、逼近论、凸分析、算法复杂度理论等多门学科。专门研究计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，重新组织已有的知识结构使之不断改善自身的性能

  • 机器学习源于人工智能(Artificial Intelligence,AI)领域，是使计算机具有智能的根本途径，其应用遍及人工智能的各个领域，它主要使用归纳、综合而不是演绎

  • 机器学习所研究的主要内容，是关于在计算机上从数据中产生模型(model)的算法，即学习算法(learning algorithm)

定义

  • Arthur Samuel(1959)：一门不需要通过外部程序指示而让计算机有自我学习的学科

  • Langley(1996)：机器学习是一门人工智能的科学，该领域的主要研究对象是人工智能，特别是如何在经验学习中改善具体算法的性能

  • Tom Michell(1997)：针对经验E(experience)和一系列的任务T(task)和一定表现的衡量P(performance)，如果随着经验E的积累，针对定义好的任务T可以提高表现P，就说计算机具有学习能力

常见符号

  • m：训练样例的数量

  • X’s：输入(input)变量，特征(feature)或属性(attribute)

  • 属性上的取值，成为属性值(attribute value)

  • 一个示例也可以称为一个特征向量(feature vector)

  • y’s：输出(output)变量,目标(target)或标记(label)

  • d：样本的维数(dimensionality)

  • (x,y)：一个训练样例

  • (x(i),y(i))：训练集中第i个样例

  • 属性空间(attribute space)：属性张成的空间，或称样本空间(sample space)

  • 标记空间(label space)：所有标记的集合，或称输出空间(output space)

  • 假设空间(hypothesis space)：学习过程进行搜索的空间

  • 版本空间(verison spave)：由于学习过程是基于有限样本训练集进行的，因此，可能有多个假设与训练集一致，即存在着一个与训练集一直的假设集合，称之为样本空间

  • 归纳偏好(inductive bias)：机器学习算法在学习过程中对某种类型假设的偏好

  • 奥卡姆剃刀(Occam’s Razor)是一种常用的、自然科学研究中最基本的原则，若有多个假设与观察一致，则选择最简单的那个

  • 泛化(generalization)：学得模型适用于新样本的能力，称为泛化能力，具有强泛化能力的模型能很好地适用于整个样本空间

分类

  • 监督学习(Supervised learning)：训练集有类别标记(class label)

  • 非监督学习(Unsupervised learning)：训练集没有类别标记

  • 半监督学习(Semi-supervised learning)：训练集由有类别标记和无类别标记组成

  • 强化学习(Reinforcement learning)

  • 推荐系统(Recommender systems)

发展历程

  • 二十世纪五十年代到七十年代初-推理期

  • “只要赋予机器逻辑推理能力，机器就具有智能”

  • 二十世纪五十年代中后期，基于神经网络的连接主义(connectionism)学习开始出现

  • 二十世纪六七十年代，基于逻辑表示的符号主义(symbolism)学习技术、以决策理论为基础的学习技术和强化学习技术(reinforcement learning)蓬勃发展

  • 二十世纪七十年代中期开始-知识期

  • “要使机器具有智能，就必须设法使机器拥有知识”

  • 二十世纪八十年代-机器学习成为一个独立的科学领域、各种机器学习技术百花初绽的时期

  • R.S.Michalski等人把机器学习研究划分为：从样例中学习、在问题求解和规划中学习、通过观察和发现学习、从指令中学习等种类

  • E.A.Feigenbaum等人则把机器学习分为：机械学习、式教学系、类比学习和归纳学习

  • 从样例中学习的一大主流是符号主义学习，其代表包括决策树(decision tree)和基于逻辑的学习

  • 其中基于逻辑的学习的著名代表是归纳逻辑程序设计(Inductive Logic Programming，简称ILP)

  • 在二十世纪九十年代中期前，从样例中学习的另一主流技术是基于神经网络的连接主义学习

  • 其中BP（反向传播算法）一直是被应用的最广泛的机器学习算法之一

  • 二十世纪九十年代中期-统计学习(statistical learning)登场并迅速占据主流舞台

  • 代表性技术是支持向量机(Support Vector Machine，简称SVM)以及更一般的核方法(kernel methods)

  • V.N.Vapnik和A.J.Chervonenkis在1974年提出了结构风险最小化原则

  • 二十一世纪初-连接主义学习卷土重来，掀起以深度学习为名的热潮

  • 所谓的深度学习，狭义的说就是很多层的神经网络

应用现状

  • 在计算机视觉、自然语言处理等计算机应用技术领域，机器学习已成为最重要的技术进步源泉之一

  • 自动驾驶、语音识别、语言翻译

  • 数据分析同样是机器学习技术的舞台，各种机器学习技术已经在这个舞台上大放异彩

  • 推荐系统、基因识别
