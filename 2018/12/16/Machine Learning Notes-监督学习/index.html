监督学习

定义

- 监督学习是机器学习中的一个方法，可以由训练集中学到或者建立一个模式（函数/learning model），并依此模式推测新的实例

- 训练资料是由输入物件（通常是向量）和预期输出所组成

- 函数的输出可以是一个连续的值（称为回归），或是预测一个分类标签（称作分类）

回归

- 回归(Regression)：目标标记为连续性数值(continuous numeric value)  

<div align=center>![Regression](/Machine Learning Notes-监督学习/Regrssion.jpg)

- 假设我们希望通过给定的房屋面积来预测其市场价格，此时该房屋的面积就是特征(feature)，房屋的售价就是目标(target)

- 首先，我们的数据不仅包含房屋的面积，还包含其相对应的价格，我们所要做的就是通过面积来预测房价，这样具有监督学习的特点

- 其次，我们的输出数据-房价，可以看作是连续的值，所以这个问题是一个回归问题

分类

- 分类(Classfication)：目标标记为类别型数据(category value)

<div align=center>![Classification](/Machine Learning Notes-监督学习/Classfication.jpg)

- 假设我们希望利用现有的乳腺癌数据去预测任意乳腺肿瘤为良性还是恶性，医疗数据以肿瘤的大小以及该肿瘤是良性还是恶性组成，可以规定0代表良性，1代表恶性，这样的问题属于分类的范畴，因为我们的目标输出为一个离散值
