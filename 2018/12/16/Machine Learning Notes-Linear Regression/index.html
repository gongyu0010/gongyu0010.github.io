线性回归

- 线性回归可以分为：单变量线性回归(Univariate linear regression)和多变量线性回归(Multiple Variables)

单变量线性回归

定义

- 一般的，令D={x<sub>1</sub>,x<sub>2</sub>...x<sub>m</sub>}表示包含m个实例的数据集，每个示例由d个属性描述，则每个示例x<sub>i</sub>=(x<sub>i1</sub>;x<sub>i2</sub>;...x<sub>id</sub>)是d维样本空间X中的一个向量，其中x<sub>ij</sub>是x<sub>i</sub>是第i 个示例在第j个属性上的取值

- 特别的，当属性的数量为1时，称之为单变量线性回归

假设函数

- 假设函数(Hypothesis Function)-h<sub>θ</sub>

- 使用某种学习算法对训练集的数据进行训练，我们可以得到特征同标记之间的某种关系，即假设函数


- 在预测房价的例子中，假设函数就是一个房价关于面积的函数，学习算法的目标就是为了得到这个函数，这样在此之后，给定一个房子的面积，我们就可以对其房价进行预测
