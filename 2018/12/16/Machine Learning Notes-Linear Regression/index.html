线性回归

  • 线性回归可以分为：单变量线性回归(Univariate linear regression)和多变量线性回归(Multiple Variables)

单变量线性回归

定义

  • 一般的，令表示包含m个实例的数据集，每个示例由d个属性描述，则每个示例是d维样本空间X中的一个向量，其中是是第i 个示例在第j个属性上的取值

  • 特别的，当属性的数量为1时，称之为单变量线性回归

假设函数

  • 假设函数(Hypothesis Function)-

  • 使用某种学习算法对训练集的数据进行训练，我们可以得到特征同标记之间的某种关系，即假设函数

  • 在预测房价的例子中，假设函数就是一个房价关于面积的函数，学习算法的目标就是为了得到这个函数，这样在此之后，给定一个房子的面积，我们就可以对其房价进行预测

代价函数

  • 在得到了假设函数之后，就可以利用相关数据进行预测了

  • 以预测房价为例，对于假设函数中的两个未知变量、，选择的值不同，那么模型的泛化能力也就不同，下图展示了三种不同和情况下所对应的假设函数

  • 为了选择泛化能力更强的和组合，可以利用均方误差(MSE)作为衡量假设函数的泛化能力的标准，即要求每个样例的估计值与真实值之间的差的平方的均值最小，此时均方误差就称作预测房价模型的代价函数，而代价函数的最优值所对应的和被称为优化目标

梯度下降法

  • 在向量微积分中，标量场的梯度是一个向量场，标量场中某一点的梯度指向这点标量场增长最快的方向（固定方向的长度）

  • 梯度的绝对值是长度为1的方向重函数最大的增长率，特别的，一元函数的梯度就是导数

  • 梯度下降法(Gradient Descent)是一个一阶最优化算法，通常也称为最速下降法

  • 要使用梯度下降法去找一个函数的局部最小值，必须向函数上当前点对应梯度（或者近似梯度）的反方向的规定步长距离点进行迭代搜索

  • 如果相反地向梯度正方向迭代进行搜索，则会接近函数的局部极大值点（这个过程被称为梯度上升法）

  • 可以将梯度下降法同下山联系起来，如果想要尽可能快的下坡，应该每次都往坡度最大的方向下山

  • 梯度下降法得到的结果同初始状态相关，当从不同的点开始时，可能会达到不同的局部极小值（陷入局部最优解）

  • 局部最优解，是因为不同的起始点导致的算法进入了不同的局部最低点

  • 但是，由于单变量线性回归的代价函数总是一个凸函数(Convex Function)，这样的函数没有局部最优解，只有全局最优解，所以在使用梯度下降法优化单变量线性回归问题时，总会得到一个全局最优解

梯度与学习率

  • 为了研究梯度下降法是如何帮助我们找到最优解，我们设，假设一开始选取的在最低点的右侧，此时的梯度（斜率）是一个正数，从图中我们可以看出，当使用梯度下降法更新的时候，它的值会减小，慢慢逼近最低点

  • 同理，假设一开始选取的在最低点的左侧，此时的梯度是一个负数，使用梯度下降法更新时，它的值会增大，即从另一个方向逼近最低点

  • 如果一开始选取的恰好在最适位置，那么更新时，它的值不会发生变化

  • 学习率会直接影响每一次梯度下降的幅度，如果太小，那么的值每次变化的也会很小，那么梯度下降的就会非常慢，相应的如果太大，那么的值每次变化的会很大，梯度下降的就会非常快

  • 梯度下降的过慢，会导致算法优化的时间损耗增加，但是梯度下降的过快，有可能导致在梯度下降的过程中，越过了最低点，甚至永远没有办法到达最低点（发散状态）

  • 随着越来越接近最低点，相应的梯度（绝对值）也会逐渐减小，所以每次下降的幅度也会逐渐减小，所以我们不需要改变学习率的值
